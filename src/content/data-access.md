# Data Access Policy

This policy outlines how you can access the data stored in our package registry.

## Public Data

All public package information, such as package names, versions, descriptions, and download statistics, is available through our public API. Please refer to our API documentation for details on how to access this data.

## Private Data

Your private user data, such as your email address and other account details, is not shared publicly. You can access and manage your personal data through your user profile page.

## Data Dumps

We do not currently provide full data dumps of the registry. If you have a specific need for bulk data access, please contact us to discuss your use case.

## Data Portability

You can request an export of your personal data by contacting us at `support@example.com`.
